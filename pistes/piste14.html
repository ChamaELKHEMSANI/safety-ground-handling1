BRISER LES SILOS, PARTAGER LES BONNES PRATIQUES

Réseau de partage d'expériences, base REX commune, achats et formations groupés.

CDG, c'est 20+ entreprises de handling qui travaillent côte à côte sur la même plateforme, avec les mêmes avions, les mêmes pistes, les mêmes risques. Pourtant, chacun travaille dans son silo. Une bonne pratique développée par l'une reste confinée. Un accident subi par l'une est subi par une autre 6 mois plus tard. Les investissements et formations sont doublés. La quatorzième piste propose de briser ces silos pour multiplier par 20 l'intelligence collective disponible.

1. PROBLÉMATIQUE ADRESSÉE

Constat fondamental : CDG, c'est 20+ entreprises de handling qui travaillent côte à côte sur la même plateforme, avec les mêmes avions, les mêmes pistes, les mêmes risques.

Paradoxe :
  • Les conducteurs se croisent quotidiennement sur le tarmac
  • Ils partagent les mêmes difficultés, les mêmes frustrations
  • Mais leurs retours d'expérience ne sont jamais partagés
Notre hypothèse : La sécurité n'est pas un sujet de compétition. C'est un sujet de coopération.

Ce qui est bon pour un handler est bon pour tous. Ce qui est dangereux pour l'un est dangereux pour tous.

Briser les silos, c'est multiplier par 20 l'intelligence collective disponible.

11. CONDITIONS DE RÉUSSITE

Facteurs clés

  1. Engagement direction : Participation inscrite dans les objectifs des responsables sécurité
  2. Confidentialité : Charte de non-divulgation, confiance mutuelle
  3. Utilité perçue : Sujets concrets, pas de réunions "pour rien"
  4. Bienveillance : Zéro jugement, zéro hiérarchie
  5. Équité : Grands et petits handlers sur un pied d'égalité
  6. Pérennité : Calendrier annuel fixe, rituel

Pièges à éviter

  • Réunionite : 2h30 max, ordre du jour serré
  • Jargon : Parler simple, pas de sigles incompréhensibles
  • Monologue : Animer, ne pas subir
  • Entreprise "professeur" : Tous experts, tous apprenants
  • Absence de suivi : Compte-rendu + plan d'action
